# oqb-printer
A userscript to print and download edcity.hk OQB assessment.
## Installation
1. Install a userscript engine (e.g. [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)) to your browser.
2. [Allow user scripts](https://www.tampermonkey.net/faq.php?locale=en#Q209) in the extension settings.
3. Install [this userscript](https://raw.githubusercontent.com/kenkoedu/oqb-printer/main/oqb-printer.user.js)
